# SacarDeterminanteMatriz

Programa que obtiene la Determinante de una matriz `3x3` (únicamente)

### Ejemplo:
||||
| :---: | :---: | :---: |
| 2 | 0 | 3 |
| -1 | 3 | 5 |
| -2 | 1 | 4 |

La determinante de la matriz anterior es `29` siguiendo las siguientes ecuaciones:
